## 部署模式

### 边缘端部署 (Edge Deployment)
- 适用于低延迟、隐私敏感场景
- 本地处理音频和基础意图识别
- 复杂决策可选择性上云

### 云端部署 (Cloud Deployment)
- 适用于算力受限设备
- 完整的AI能力支持
- 跨设备协同处理

### 混合部署 (Hybrid Deployment)
- 边缘预处理 + 云端深度分析
- 兼顾延迟和能力

## 核心模块

| 模块 | 职责 | 关键能力 |
|------|------|----------|
| 音频处理 | 音频信号处理与特征提取 | VAD、ASR、说话人识别、情绪识别 |
| 场景分析 | 对话场景判断 | 人机/人人对话识别、环境感知 |
| 意图引擎 | 用户意图理解与决策 | NLU、多轮对话、设备能力匹配 |
| 设备管理 | 设备注册与能力管理 | 设备发现、能力抽象、状态同步 |
| 决策引擎 | 执行决策与响应生成 | 主动询问、自动执行、响应合成 |

## 数据流

```
    │          │          │          │          │          │
    └──────────┴──────────┴──────────┴──────────┴──────────┘
                        上下文管理 (Context Manager)
```
