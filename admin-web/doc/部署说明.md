# 知境中枢 - 管理端部署说明

## 技术栈

- Vue 3.3+
- TypeScript 5+
- Vite 5+
- Element Plus 2.4+
- Pinia
- Vue Router

## 环境要求

- Node.js 20+
- pnpm

### 安装 Node.js

推荐使用 nvm 安装：

```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash

# 重新加载 shell 配置
source ~/.bashrc

# 安装 Node.js 20
nvm install 20

# 验证版本
node -v   # 应显示 v20.x.x
```

### 安装 pnpm

```bash
npm install -g pnpm
```

## 部署步骤

1. 进入前端目录
```bash
cd admin-web
```

2. 安装依赖
```bash
pnpm install
```

3. 构建生产版本
```bash
pnpm build
```

4. 运行生产版本
```bash
pnpm preview --host --port 3000
```

访问 `http://服务器IP:3000`
