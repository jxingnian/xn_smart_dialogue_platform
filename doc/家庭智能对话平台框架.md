# 家庭智能对话平台框架

## 平台架构

```
                    +------------------+
                    |      管理端       |
                    |  Web (私有/公网)  |
                    +--------+---------+
                             |
                             v
+------------------------------------------------------------+
|                          大脑端                             |
|                   (云端部署/私有部署)                        |
|                                                            |
|    +----------------+                                      |
|    |     记忆层      |                                      |
|    | 短期    |  长期  |                                      |
|    +---+----+---+---+                                      |
|        |        |                                          |
|      写入     读/写                                         |
|        |        |                                          |
|        v        v                                          |
|    +------+  +---------------------------+                 |
|    |感知层|->|          决策层            |                 |
|    +------+  |  +------+      +------+   |                 |
|              |  | 小脑  |      | 大脑  |   |                 |
|              |  |<100ms|      |1-10s |   |                 |
|              |  +--+---+      +--+---+   |                 |
|              |     +-----+------+       |                 |
|              |           v              |                 |
|              |       响应仲裁            |                 |
|              +------------+-------------+                 |
|                           |                               |
+---------------------------+-------------------------------+
                            |
                            v
                    +-------+--------+
                    |     设备端      |
                    | 输入/输出/复合  |
                    +----------------+
```

---

## 数据流向

```
设备数据 --> 感知层 --> 写入短期记忆
               |
               v
         +-----+-----+
         |           |
         v           v
       小脑        大脑 <-- 读取记忆
      (<100ms)   (1-10s)
         |           |
         +-----+-----+
               |
               v
           响应仲裁
         (快先慢覆盖)
               |
               v
           设备执行
```

---

## 三个子项目

| 子项目 | 说明 |
|-------|------|
| 设备端 | 输入设备/输出设备/复合设备，类似人体器官 |
| 大脑端 | 感知层+决策层(小脑/大脑)+记忆层，可云端或私有部署 |
| 管理端 | Web界面，支持私有访问和公网访问 |
